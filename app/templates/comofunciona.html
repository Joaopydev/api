Guia da API

Controlar sua finanças se tornou ainda mais fácil, por isso nós desenvolvemos a API Finance Professional. Uma API para desenvolvedores controlarem seus gastos mensais e assim se organizarem financeiramente. Nessa sessão você verá todas as funcionalidades que a nossa API possui. Tenha em mente que será essencial o seu token que é criado automaticamente, visualize-o: Seu Token

Sessão: Adicionar Receita.

Receitas API de Receitas: Gerencie todas as suas fontes de renda com eficiência. Registre, atualize, consulte e exclua receitas financeiras, mantendo um controle claro e organizado de todas as suas entradas de dinheiro. Aprenda a utilizar com a sessão ao lado.
Para adicionar suas receitas, você deve enviar para a API um dicionário com os seguintes dados:
{"user": "E-mail da sua conta",
"token": "Seu token de usuário",
"tipo" : "Tipo de receita (Ex: Trabalho )",
"valor" : "Valor relacionado a sua receita"}
Para adicionar suas despesas, você deve enviar para a API um dicionário com os seguintes dados:
{"user": "E-mail da sua conta",
"token": "Seu token de usuário",
"tipo" : "Tipo de despesa (Ex: Academia )",
"valor" : "Valor relacionado a sua despesa"}

Sessão: Adicionar Despesa.

As despesas incluem qualquer transação ou saída de fundos que reduza o saldo disponível do usuário. Elas podem ser categorizadas de várias maneiras, permitindo uma análise detalhada e uma gestão financeira eficiente.
Exemplos de despesas: Alimentação, moradia, transporte, saúde e etc...

Sessão: Obter Resumo Financeiro.

O resumo financeiro oferece uma análise agregada das transações financeiras de um usuário em um período especificado. Ele inclui informações sobre o total de receitas, o total de despesas, o saldo líquido e outras métricas financeiras relevantes. Este recurso é projetado para fornecer uma visão clara e instantânea da saúde financeira do usuário.
Para obter o seu resumo financeiro, você precisa apenas copiar o link acima. E alterar "SEU_MAIL" colocando o seu e-mail de usuário e "SEU_TOKEN" colocando o seu token de usuário.
A API vai retornar todos os dados financeiros referente as receitas e despesas do usuário.
Para remover suas receitas, você deve enviar para a API um dicionário com os seguintes dados:
{"user": "E-mail da sua conta",
"token": "Seu token de usuário",
"id_receita" : "ID da receita que deseja remover"}
Obs: O ID da receita você consegue visualizar no seu resumo financeiro.

Sessão: Remover Receitas do usuário.

A remoção de receitas é uma operação crítica para garantir que os registros financeiros do usuário reflitam com precisão suas transações reais. Ao permitir a exclusão de receitas, a API ajuda os usuários a corrigirem entradas erradas e a manterem seus registros financeiros atualizados.
